Classpublic class MenuPanelDisplayContext
InheritanceMenuPanelDisplayContext Object
Implements com.custardbelly.as3flobile.util.IObjectPool

MenuPanelDisplayContext represents a model for configuration of an IMenuPanelDisplay instance. By extending IObjectPool, this model also acts as a factory for creating new instances based on configuration properties.
